# 🎬 Netflix Clone (HTML & CSS Only)

A fully responsive, pixel-perfect front-end clone of the Netflix landing and browse pages. Built completely from scratch using semantic HTML5 and modern CSS3 techniques to master responsive layout design.

## 🌟 Key Features
*   **Pixel-Perfect UI:** Closely mimics the actual Netflix dark theme, layout structures, and iconic branding elements.
*   **100% Responsive Design:** Smooth, fluid layouts that adapt seamlessly across mobile phones, tablets, and large desktop screens using CSS Media Queries.
*   **Interactive CSS Components:** Features pure CSS dropdowns, smooth hover animations on movie cards, and a functional accordian-style FAQ section using HTML `<details>` and `<summary>` tags.
*   **Optimized Performance:** Fast load times by using SVG icons and clean, lightweight native CSS code without heavy framework overhead.

## 🛠️ Built With
*   **HTML5:** Semantic structuring (`<nav>`, `<header>`, `<section>`, `<main>`, `<footer>`) for accessibility.
*   **CSS3 Custom Properties:** Used CSS variables for global consistency across colors (Netflix Red, Dark Backgrounds) and typography.
*   **Flexbox & CSS Grid:** Leveraged Flexbox for alignment and navigation components, and CSS Grid to handle complex, responsive movie poster rows.

## 📱 Responsive Breakpoints Used
*   `Desktop / Large Screens:` >= 1024px (Multi-column grids, full navigation bar layout)
*   `Tablets:` 768px - 1023px (Scaled down rows, condensed paddings)
*   `Mobile Devices:` <= 767px (Single-column layout, compact headers, touch-friendly tap targets)
